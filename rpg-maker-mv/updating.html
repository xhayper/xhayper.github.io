# Updating

This section will teach you how to upgrade RPG Maker MV's component.

# NW.js

NW.js allow you to use NodeJSopen in new window module inside DOMopen in new window, It's used by RPG Maker MV to... well... run the game...

RPG Maker MV came with NW.js 0.29.4open in new window and Node 9.11.1open in new window

By upgrading NW.jsopen in new window, you also upgrade NodeJSopen in new window

WARNING

Make sure to backup! If you messed up, you have to reinstall RPG Maker MV.

# Test
  1. Head to NW.js websiteopen in new window and download the SDK version.
  2. Navigate to RPG Maker MV's folder.
  3. Drag everything inside the downloaded zip into the nwjs-win-test folder.
  4. Rename from nw.exe to Game.exe.
# Deploy
  1. Head to NW.js websiteopen in new window and download the NORMAL version.
  2. Navigate to RPG Maker MV's folder.
  3. Drag everything inside the downloaded zip into the nwjs-win folder.
  4. Rename from nw.exe to Game.exe.

# Pixi

Pixiopen in new window is the rendering engine used by RPG Maker MV.

RPG Maker MV came with Pixi 4.5.4open in new window

TIP

If you don't want to go through the trouble, here's the pre-made one (Only script, Pixi 6.2.1).

WARNING

Make sure to backup! Upgrading Pixi is a very VERY hard process.

It involve modifying core script and modifying library files.

It can also break complatibility with some plugins, but that rarely happened (from FeniXEngine MV's guideopen in new window).

# Instructions

  1. Upgrade NW.js
  2. Follow FeniXEngine MV's PIXI 4.5.4 to PIXI 5.3.0 upgrade guideopen in new window.
  3. Download Pixi from Githubopen in new window and replace the old one (located inside root/js/lib).
  4. Download Pixi Tilemap from Githubopen in new window (Download source, and it's located in dist folder) and replace the old one (located inside root/js/lib/).
  5. The latest version of Pixi Tilemap doesn't include the function that RPG Maker MV use so we have to download the rpgmaker.jsopen in new window from the demo folder and put it in root/js/lib/.
  6. Open up index.html in your project root and
<!-- Replace from -->
<script type="text/javascript" src="js/libs/pixi-tilemap.js"></script>
<!-- To -->
<script type="text/javascript" src="js/libs/pixi-tilemap.js"></script>
<script type="text/javascript" src="js/libs/rpgmaker.js"></script>
